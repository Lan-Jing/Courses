# [Loop] Power

# Description
 You should write a program to calculate the power of 3.   
For example, given a positive integer n, you should calculate 3^n.   
 
# Input
The input contains a single positive integer n(0 < n <= 200)
 
# Output
 An integer.
 
# Sample Input
```
48
```
# Sample Output
```
79766443076872509863361
```
