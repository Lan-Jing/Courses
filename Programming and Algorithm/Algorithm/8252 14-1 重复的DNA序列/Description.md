# 14-1 重复的DNA序列

## 问题描述:<br>
所有 DNA 都由一系列缩写为 A，C，G 和 T 的核苷酸组成，例如：“ACGAATTCCG”。在研究 DNA 时，识别 DNA 中的重复序列有时会对研究非常有帮助。<br>
编写一个函数来查找 DNA 分子中所有出现超过一次的 10 个字母长的序列（子串）。<br>
输入格式：<br>
一行DNA序列。<br>
输出格式：<br>
所有重复的子串。
<br>

## 输入样例<br>
AAAAACCCCCAAAAACCCCCCAAAAAGGGTTT<br>

## 输出样例<br>
AAAAACCCCC<br>
CCCCCAAAAA<br>
