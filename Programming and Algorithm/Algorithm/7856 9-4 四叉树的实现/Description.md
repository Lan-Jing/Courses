# 9-4 四叉树的实现

<br>四叉树将二维空间划分成四个象限，每一个节点最多有四个子节点，表示左上、右上、左下、右下四个子区域，平均划分了父节点的区域范围，每个子节点只包含一个数据。
<br>给定一个二维空间范围在x∈[0,10]，y∈[0,10]，给定若干个点，请构造一个四叉树并按照层序遍历的顺序输出每个点。
<br>输入包含两个部分，第一个部分K表示K个点，第二部分接下来K行表示每个点的坐标，在限定的空间范围内。
<br>输出K行为层序遍历的结果。
<br>输入
<br>7 
<br>2 5 
<br>3 7.5 
<br>4.2 6.8 
<br>5 2.5 
<br>8 5.6 
<br>7 3.5 
<br>5.5 3.5 
<br>输出
<br>8 5.6
<br>2 5
<br>5 2.5
<br>3 7.5
<br>4.2 6.8
<br>5.5 3.5
<br>7 3.5
